# myMigScan
Shopping by Smartphone scan &amp; pay

myMigScan macht das digitale Einkaufen mit dem Smartphone möglich und substituiert somit den Scanner im Einkaufsladen.
Ausbaustufe ist der Verzicht des Smartphones als Zwischenlösung mit einer Virtual Reality Lösung.

Einkaufen in der virtuellen Welt comes true.
